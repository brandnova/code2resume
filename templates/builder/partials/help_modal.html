How to Use Code2Resume

Press ? to toggle

Workspace & Editor

Split-Pane Layout HTML/CSS tabs on the left, live preview iframe on the right. Updates instantly on every keystroke.
Code Editors Powered by CodeJar + Highlight.js. Supports Tab-to-indent, native Undo/Redo, and all browser text shortcuts.

Controls & Export

Framework Injector Select None, Tailwind, or Bootstrap. CDN injects directly into preview & PDF engine.
Paper Size Choose A4, Letter, Legal, or A5. Preview resizes to match 96 dpi dimensions.
Page Break Guides Toggle dashed overlay lines to prevent awkward content splits across pages.
Export PDF Renders pixel-perfect output via headless Chromium (Playwright) with your exact framework & paper settings.

Persistence & Themes

Auto-Save State saves to Django session every 1.5s. Refreshing or reopening restores your last work.
Dark/Light Theme Toggle via sun/moon button in navbar. Preference persists in localStorage. Highlight.js swaps automatically.
Built for developers who care about the source code of their career. Press Ctrl+K for full keyboard shortcuts.
