        Features
      

      
Everything a developer needs.
Nothing they don't.

      
Built with the same discipline you'd bring to a production codebase. No bloat, no lock-in.

    


    

      

        

        
Split-Pane Editor

        
HTML and CSS side-by-side with CodeJar syntax highlighting. Preview updates on every keystroke.

      


      

        

        
Framework Injector

        
Drop in Tailwind CSS or Bootstrap with a single dropdown. The CDN is injected instantly.

      


      

        

        
Playwright PDF Engine

        
Headless Chromium renders your exact HTML to PDF. What you see is what you get.

      


      

        

        
Page Break Guides

        
Visual A4/Letter/Legal/A5 boundary overlays in the preview so you never clip text at print time.

      


      

        

        
Auto Session Save

        
Your work is saved to the server every 1.5 seconds. Reload the page and your resume is still there.

      


      

        

        
Cloud Save

        
Sign in with Google or GitHub. Save multiple resumes and open them from any device.

      

    

  


  
  

    

      

        workflow
      

      
Four steps. One great resume.

    

    
    

      

        
01

        
Open workspace

        
No signup required. Start with the default template or paste your own HTML.

      

      

        
02

        
Write HTML & CSS

        
Full control. Use utility classes, custom fonts, grid layouts, anything a browser can render.

      

      

        
03

        
Preview live

        
The live iframe reflects your exact resume as you type. Toggle page guides to check boundaries.

      

      

        
04

        
Export as PDF

        
Click Export. Playwright renders your HTML in headless Chromium and returns a pixel-perfect PDF.
