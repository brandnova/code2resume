// features

      
Everything a developer needs.
Nothing they don't.

      
Built with the same discipline you'd bring to a production codebase. No bloat, no lock-in, no surprises.

    


    


      

        

        
Split-Pane Editor

        
HTML and CSS side-by-side with CodeJar syntax highlighting. Preview updates on every keystroke with zero latency.

      


      

        

        
Framework Injector

        
Drop in Tailwind CSS or Bootstrap with a single dropdown. The CDN is injected into your preview iframe instantly.

      


      

        

        
Playwright PDF Engine

        
Headless Chromium renders your exact HTML to PDF. What you see in the preview is exactly what you get in the file.

      


      

        

        
Page Break Guides

        
Visual A4/Letter/Legal/A5 boundary overlays in the preview so you never clip a line of text at print time.

      


      

        

        
Auto Session Save

        
Your work is saved to the server every 1.5 seconds as you type. Reload the page and your resume is still there.

      


      

        

        
Account & Cloud Save

        
Sign in with Google or GitHub. Save multiple named resumes and open them from any device at any time.

Four steps. One great resume.

    

    

      

        
01

        
Open the workspace

        
No signup required. Start with the default template or paste your own HTML instantly.

      

      

        
02

        
Write your HTML & CSS

        
Full control. Use utility classes, custom fonts, grid layouts — anything a browser can render.

      

      

        
03

        
Preview in real-time

        
The live iframe reflects your exact resume as you type. Toggle page guides to check boundaries.

      

      

        
04

Click Export. Playwright renders your HTML in headless Chromium and returns a pixel-perfect PDF.
